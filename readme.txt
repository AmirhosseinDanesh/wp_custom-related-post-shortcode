This plugin adds a shortcode to display related posts by their ID.

== Description ==

Custom Related Post Shortcode is a simple plugin that allows you to display related posts anywhere in your content by using a shortcode and providing the post ID.

**Features:**
- Display a related post by ID.
- Customize the position of related posts in your content.
- Lightweight and easy to use.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-related-post` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `[related_post id="123"]` shortcode in your posts or pages, replacing `123` with the ID of the post you want to display.

== Frequently Asked Questions ==

= How do I find the post ID? =
You can find the post ID by going to the post in the WordPress dashboard. The ID will be in the URL of the edit screen (e.g., `post=123`).

= Can I display more than one related post? =
Yes, you can add multiple shortcodes to display different posts.
